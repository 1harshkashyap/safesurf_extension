## 🧠 Overview

**SafeSurf Extension** is a browser-based security tool designed to **protect users from unsafe, malicious, and inappropriate web content in real time**.

Instead of relying only on static blocklists, SafeSurf:

* Dynamically analyzes web content
* Detects harmful patterns
* Alerts or blocks users before damage occurs

> ⚡ Built for real-world browsing safety — lightweight, fast, and intelligent.

---

## 🔥 Features

* 🛡️ **Real-Time Website Protection**
  Scans and evaluates websites instantly

* 🤖 **AI-Based Content Detection**
  Identifies phishing, unsafe, or harmful content

* 🚫 **Smart Blocking & Redirection**
  Prevents access to risky pages

* 🔔 **User Alerts & Notifications**
  Warns users before entering dangerous sites

* 📊 **Usage Insights (if implemented)**
  Tracks browsing behavior and safety levels

## ⚙️ Installation

### 🔹 Load Extension (Manual)

1. Clone the repository

```bash
git clone https://github.com/1harshkashyap/safesurf_extension.git
```

2. Open Chrome and go to:

```
chrome://extensions/
```

3. Enable **Developer Mode**

4. Click **Load unpacked**

5. Select the project folder

---

## 🚀 How It Works

1. User visits a website
2. SafeSurf analyzes page content & URL
3. System classifies risk level
4. If unsafe:

   * ⚠️ Shows warning
   * 🔁 Redirects or blocks access
