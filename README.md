## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/ARGUS_X.git
cd ARGUS_X
```

### 2. Backend Setup

```bash
cd backend
pip install -r requirements.txt
python main.py
```

### 3. Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## 🐳 Docker Deployment (Recommended)

```bash
docker build -t argus_x .
docker run -p 8000:8000 argus_x
```
